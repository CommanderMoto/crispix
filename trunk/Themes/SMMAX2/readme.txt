1. About
************************************************************

/!\ for use with Stepmania Beta 6/!\

"SMMAX2" is a skin created to add a new and P-RESH appearance to Stepmania. The creation is entirely original, and any resemblance to the commercial game "Dance Dance Revolution" is ENTIRELY COINCIDENTAL. The skin is pretty much the final version. Check the DDRManiaX boards and the Stepmania website for updates, if any at all.

************************************************************
2. Applying the Skin
************************************************************

Execute the smzip to the proper Stepmania folder then select "SMMAX2" as the theme in
"Appearance Options" in StepMania.
